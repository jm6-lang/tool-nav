## 联系我们

感谢您使用 AI工具导航！如果您有任何问题、建议或想要推荐优质工具，欢迎通过以下方式联系我们。

### 📧 邮箱联系

- **通用咨询**：contact@skillxm.cn
- **工具收录**：submit@skillxm.cn

我们会在收到邮件后 48 小时内回复。

### 💻 GitHub

如果您是开发者或技术爱好者，欢迎通过 GitHub 与我们互动：

- **提交工具收录建议**：[创建 Issue](https://github.com/jm6-lang/tool-nav/issues/new?template=tool-submission.md)
- **报告 Bug 或问题**：[提交 Issue](https://github.com/jm6-lang/tool-nav/issues/new)
- **贡献代码**：[提交 Pull Request](https://github.com/jm6-lang/tool-nav/pulls)

### 📋 工具收录标准

我们欢迎推荐优质工具，收录标准如下：

1. **实用性** — 真正能提升效率，有实际使用价值
2. **稳定性** — 服务稳定，有持续维护和更新
3. **安全性** — 无恶意软件，无隐私泄露风险
4. **易用性** — 上手简单，界面友好
5. **合法性** — 遵守相关法律法规
