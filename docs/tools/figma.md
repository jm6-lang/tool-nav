## 工具简介

Figma 是全球最流行的在线 UI 设计工具，彻底改变了设计行业的协作方式。与传统的 Sketch、Adobe XD 等桌面端设计工具不同，Figma 完全运行在浏览器中，支持多人实时协作编辑同一个设计文件，团队成员可以同时查看和修改设计，所有更改实时同步。Figma 提供强大的组件系统、自动布局、设计令牌（Design Tokens）等功能，支持插件扩展生态。设计师用它完成界面设计，前端开发者用它查看标注和导出资源，是设计到开发交付流程的核心工具。

## 核心功能

**实时多人协作** — 多人同时编辑同一设计文件，光标位置和操作实时可见，内置评论和反馈系统，设计评审无需反复发送文件。
**强大的组件系统** — 支持创建可复用的组件和变体，修改主组件时所有实例自动更新，配合自动布局（Auto Layout）实现响应式设计。
**开发者模式** — 提供专门的开发者模式，前端开发者可以查看精确的尺寸、间距、颜色值、CSS 代码等标注信息，直接复制代码片段。

## 适用场景

**UI/UX 设计** — 从线框图到高保真原型，Figma 覆盖完整的设计流程，支持交互原型演示、动效设计和用户测试。
**设计系统维护** — 利用组件库和设计令牌构建和维护设计系统，确保产品各端视觉一致性，设计师和开发者共享同一套规范。

## 使用技巧

**善用自动布局** — Auto Layout 是 Figma 最强大的功能之一，掌握后可以快速创建响应式设计，修改内容时布局自动调整，大幅提升设计效率。

## 优缺点简评

**优势：** 实时协作体验优秀、组件系统强大、免费版功能够用、插件生态丰富、跨平台浏览器访问。

**不足：** 大型文件性能下降、离线功能有限、国内访问速度不稳定、免费版有项目数量限制。
