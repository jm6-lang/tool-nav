## 工具简介

Figma 是一款云端协作的 UI 设计工具，以实时多人协同编辑为核心特色。支持矢量绘图、原型交互、组件库和设计系统管理功能，设计师可以在同一个文件中实时协作。Figma 的开发者模式可以自动生成 CSS 代码和设计标注，方便前端工程师还原设计稿。Figma Community 提供海量免费的设计资源和模板，Figma 插件生态丰富。作为设计师和前端工程师协作的行业标准工具，Figma 已成为 UI/UX 设计领域的事实标准。

## 核心功能

**实时多人协作** — 多人可以在同一个设计文件中实时编辑，看到彼此的光标和操作，协作效率远超传统设计工具。

**矢量绘图与布局** — 提供强大的矢量绘图工具和 Auto Layout 自动布局功能，支持约束和响应式设计。

**组件与设计系统** — 支持创建可复用的组件和变体，构建企业级设计系统，组件更新自动同步到所有使用处。

**开发者模式** — 切换到开发者模式可以查看设计标注、间距、颜色值和自动生成的 CSS 代码，方便前端工程师还原设计。

## 适用场景

**设计团队协作** — 设计团队在 Figma 中实时协作设计，评论和反馈直接标注在设计稿上，沟通效率高。

**设计交付** — 设计师完成后切换到开发者模式，前端工程师查看标注和代码直接开发，减少沟通成本。

**设计资源复用** — 在 Figma Community 中搜索和使用免费的设计系统、图标库和模板，加速设计工作。

## 使用技巧

**使用 Auto Layout** — 尽量使用 Auto Layout 进行布局设计，这样前端工程师更容易理解布局逻辑并用 CSS/Flexbox 还原。

## 优缺点简评

**优势：** 实时协作强大、云端无需安装、组件系统完善、开发者模式实用、社区资源丰富。

**不足：** 免费版限制3个项目、需要网络连接、大型文件性能下降、中文输入偶尔有问题。
