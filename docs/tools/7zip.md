## 工具简介

7-Zip 是一款开源免费的高压缩比文件压缩工具，以其出色的压缩率和简洁高效的设计著称。支持 ZIP、RAR、7z、TAR、GZ 等数十种压缩格式的解压和创建，其中自有的 7z 格式采用 LZMA 和 LZMA2 算法，压缩率比 ZIP 高出 30-70%。支持分卷压缩、AES-256 加密和自解压格式，提供图形界面和命令行两种操作模式。软件体积仅约 1.5MB，无广告无捆绑，是 Windows 平台最常用的压缩工具。

## 核心功能

**超高压缩率** — 7z 格式采用 LZMA2 算法，压缩率比 ZIP 高出 30-70%，在压缩大文件和文本文件时优势尤为明显。

**全格式支持** — 支持解压 ZIP、RAR、7z、TAR、GZ、BZ2、XZ 等数十种格式，创建 7z、ZIP、TAR 等格式。

**AES-256 加密** — 支持 AES-256 位加密保护压缩包，加密强度高，适合保护敏感文件和重要数据。

**命令行与脚本** — 提供功能完善的命令行接口，支持在批处理脚本和自动化任务中使用，适合服务器和运维场景。

## 适用场景

**大文件传输** — 使用分卷压缩将大文件分割为多个小卷，方便通过邮件或网盘传输，接收后合并解压即可。

**敏感数据加密** — 使用 AES-256 加密压缩重要文件，设置强密码保护，确保数据安全。

**自动化运维** — 在运维脚本中使用 7-Zip 命令行进行日志压缩、备份打包和文件传输。

## 使用技巧

**使用7z格式** — 日常压缩建议使用 7z 格式，压缩率比 ZIP 高很多且支持加密，兼容性方面可用 PeaZip 或 7-Zip 打开。

## 优缺点简评

**优势：** 压缩率极高、完全开源免费、全格式支持、AES-256加密、体积小巧。

**不足：** 界面较简陋、默认关联设置激进、RAR格式只能解压不能创建、无云存储集成。
