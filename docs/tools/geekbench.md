## 工具简介

Geekbench 是由 Primate Labs 开发的跨平台基准测试软件，被广泛用于评估计算机和移动设备的处理器性能。Geekbench 6 提供单核和多核性能评分，涵盖整数运算、浮点运算和内存访问等多种测试项目。其 GPU Compute Benchmark 可评估设备的 GPU 计算能力，包括 OpenCL、CUDA 和 Vulkan 支持。测试结果可上传至在线数据库进行全球排名对比。

## 核心功能

**CPU 性能基准测试** — 通过单核和多核测试全面评估处理器性能，包含加密、图像处理、机器学习等实际应用场景的模拟测试。

**GPU 计算性能测试** — 支持 OpenCL、CUDA、Metal 和 Vulkan API，评估 GPU 在图像处理、计算摄影和机器学习等场景下的计算能力。

**跨平台对比** — 支持 Windows、macOS、Linux、Android 和 iOS，测试分数可直接跨平台对比，方便评估不同设备的相对性能。

**在线排行榜** — 测试结果可上传至 Geekbench Browser 在线数据库，查看全球设备排名和详细性能对比数据。

## 适用场景

**硬件选购参考** — 在购买新电脑或手机前，通过 Geekbench 分数对比不同设备的性能表现，做出更明智的购买决策。

**性能优化验证** — 超频玩家或系统优化者在调整硬件配置后，使用 Geekbench 量化评估性能变化幅度。

## 使用技巧

**多次测试取平均值** — 单次测试结果可能受后台程序影响，建议关闭不必要的后台应用后进行 3 次测试取平均值。

**关注子项得分** — 除了总分，还应关注各子项得分（如单核、多核、加密、浮点等），了解设备在不同场景下的具体表现。

## 优缺点简评

**优势：** 测试项目贴近实际应用、跨平台对比方便、在线数据库丰富、操作简单直观。

**不足：** 免费版功能有限（部分高级测试需付费）、跑分不能完全代表实际体验、部分场景测试负载较轻。
