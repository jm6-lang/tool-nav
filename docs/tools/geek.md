## 工具简介

Geek Uninstaller 是一款轻量级的 Windows 程序卸载工具，以彻底卸载和清理残留文件著称。与 Windows 自带的「程序和功能」相比，Geek Uninstaller 在卸载程序后会自动扫描并清理注册表残留、文件夹残留和临时文件，确保程序被完全移除。对于无法正常卸载的程序，Geek Uninstaller 还提供强制卸载功能。整个工具仅几百 KB 大小，免安装，绿色便携，是 Windows 用户管理程序的必备工具。

## 核心功能

**深度清理残留** — 卸载程序后自动扫描注册表和文件系统，找出并清理所有残留项，包括注册表键值、AppData 文件、临时文件等。
**强制卸载** — 对于损坏的安装程序或无法正常卸载的应用，Geek Uninstaller 可以强制终止相关进程并删除所有关联文件。
**绿色便携** — 整个程序仅约 3MB，无需安装，可以直接放在 U 盘中随身携带，在任何 Windows 电脑上使用。

## 适用场景

**清理流氓软件残留** — 某些软件卸载后会留下大量残留文件和注册表项，使用 Geek Uninstaller 可以彻底清理，避免系统臃肿。
**处理无法卸载的程序** — 遇到安装损坏或被锁定的程序无法正常卸载时，Geek Uninstaller 的强制卸载功能可以解决问题。

## 使用技巧

**右键使用管理员权限** — 对于系统级程序，右键 Geek Uninstaller 选择「以管理员身份运行」可以获得更完整的卸载和清理权限。
