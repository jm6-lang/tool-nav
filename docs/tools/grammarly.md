## 工具简介

Grammarly 是全球使用最广泛的 AI 英文写作助手，拥有超过 3000 万日活用户。实时检测语法错误、拼写问题、标点误用和措辞不当，提供风格优化、语气调整和清晰度建议。Grammarly 支持 Chrome 浏览器插件、桌面客户端和移动端键盘，覆盖邮件、社交媒体、文档编辑等全场景。内置抄袭检测功能，是学术论文和商务写作的质量保障工具。

## 核心功能

**实时语法检查** — 在任何输入框中实时检测语法错误、拼写问题和标点误用，支持一键修正，覆盖常见和罕见的英文语法规则。

**风格与语气优化** — 提供风格检测和语气调整建议，确保写作风格符合目标受众和场景（正式、非正式、学术等）。

**清晰度与简洁性** — 分析句子结构和用词，提供简洁化建议和替换词推荐，提升文章的可读性和表达力。

**抄袭检测** — 内置 Plagiarism Checker 功能，将文本与数十亿网页内容比对，检测潜在的抄袭问题，适合学术写作。

## 适用场景

**英文邮件写作** — 使用 Grammarly 浏览器插件在 Gmail、Outlook 等邮箱中实时检查邮件语法和语气，确保商务邮件专业得体。

**学术论文写作** — 利用语法检查和抄袭检测功能确保论文质量，避免语法错误和学术不端问题。

**非母语者英文提升** — 英文非母语的用户通过 Grammarly 的实时反馈持续改进英文写作能力，是学习英文写作的有效辅助工具。

## 使用技巧

**安装浏览器插件** — 安装 Chrome 插件后 Grammarly 会在所有网页输入框中自动工作，覆盖 Gmail、社交媒体和在线文档等场景。

## 优缺点简评

**优势：** 语法检查准确率高、浏览器插件覆盖广、抄袭检测实用、多平台支持、免费版功能够用。

**不足：** 仅支持英文、免费版功能有限（无风格和抄袭检测）、隐私争议、过度纠错偶尔影响写作风格。
