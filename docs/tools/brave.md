## 工具简介

Brave 是一款以隐私保护为核心卖点的浏览器，基于 Chromium 内核构建，兼容 Chrome 扩展。Brave 内置 Brave Shields 广告拦截和跟踪防护系统，默认阻止广告、跟踪器和恶意脚本，页面加载速度比 Chrome 快约 3 倍。Brave Search 提供不追踪用户的隐私搜索引擎，Brave Rewards 计划让用户可以选择查看隐私友好的广告并获得 BAT 代币奖励。

## 核心功能

**Brave Shields 广告拦截** — 内置强大的广告拦截和跟踪防护系统，默认阻止广告、弹窗、跟踪器和恶意脚本，无需安装额外扩展。

**Brave Search 隐私搜索** — 自研隐私搜索引擎，不追踪用户搜索历史，不建立用户画像，搜索结果独立于大型科技公司。

**Brave Rewards** — 用户可选择查看隐私友好的广告获得 BAT 代币奖励，也可以将奖励分享给常访问的网站和内容创作者。

**Chrome 扩展兼容** — 基于 Chromium 内核，兼容 Chrome Web Store 的扩展插件，可以无缝迁移 Chrome 的使用习惯。

## 适用场景

**隐私保护浏览** — 注重隐私的用户使用 Brave 的内置防护即可获得全面的隐私保护，无需安装多个隐私扩展。

**快速网页浏览** — 广告和跟踪器被默认拦截后，页面加载速度显著提升，移动端还能节省数据流量。

**隐私搜索替代** — 使用 Brave Search 替代 Google 搜索，避免搜索历史被追踪和个性化广告定向。

## 使用技巧

**开启Aggressive模式** — 在 Brave Shields 设置中选择 Aggressive 模式可以获得最强的隐私保护，但可能会影响部分网站功能。

## 优缺点简评

**优势：** 隐私保护强大、内置广告拦截、浏览速度快、Chrome扩展兼容、Brave Search隐私搜索。

**不足：** Brave Search结果质量不及Google、部分网站兼容性问题、Brave Rewards对普通用户吸引力不大。
