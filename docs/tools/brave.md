## 工具简介

Brave 是一款以隐私保护为核心特色的浏览器，由 JavaScript 创始人 Brendan Eich 创立。Brave 基于 Chromium 内核开发，完全兼容 Chrome 扩展生态，同时内置了强大的广告和追踪器拦截功能。根据官方数据，Brave 的广告拦截功能可以使网页加载速度提升 2-3 倍，并显著减少数据流量消耗。Brave 还引入了 BAT（Basic Attention Token）加密货币体系，用户可以选择查看隐私广告并获得代币奖励。

## 核心功能

**内置广告拦截** — 开箱即用的广告和追踪器拦截功能，默认拦截所有第三方广告、弹窗和恶意脚本，无需安装额外插件。
**Brave Search 搜索引擎** — 内置自研的 Brave Search 搜索引擎，不追踪用户搜索记录，不进行个性化广告投放，保护搜索隐私。
**Brave Shields 隐私控制** — 提供细粒度的隐私控制面板，用户可以按站点自定义拦截规则，管理 Cookie、指纹追踪、脚本权限等。

## 适用场景

**隐私浏览** — 日常上网时保护个人隐私，防止被广告商和追踪器收集浏览数据，适合注重隐私保护的用户。
**低配设备浏览** — 由于内置广告拦截减少了页面加载内容和 JavaScript 执行，Brave 在低配置设备上运行更流畅，内存占用更低。

## 使用技巧

**开启 Aggressive 模式** — 在 Shields 设置中选择 Aggressive 模式可以获得最强的广告拦截和隐私保护效果，但可能导致个别网站功能异常，可按站点调整。

## 优缺点简评

**优势：** 隐私保护最强、内置广告拦截、速度快、兼容 Chrome 扩展、跨平台支持。

**不足：** 部分网站兼容性问题、国内生态支持不如国产浏览器、Brave Search 中文搜索质量一般。
