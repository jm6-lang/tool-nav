## 工具简介

Vercel 是一款面向前端开发者的应用部署和托管平台，是 Next.js 框架的官方开发团队。Vercel 对 Next.js、Nuxt、SvelteKit 等前端框架提供一级支持，Git 推送自动触发构建部署，零配置即可上线。全球 CDN 边缘节点确保访问速度，提供 Serverless Functions 和 Edge Runtime 等无服务器计算能力。Vercel 的免费额度（Hobby 计划）适合个人项目和小型应用，是前端开发者首选的部署方案。

## 核心功能

**Git 自动部署** — 连接 GitHub/GitLab 仓库后，每次推送代码自动触发构建和部署，支持预览部署（Preview）和生产部署（Production）。

**全球 CDN 加速** — 部署的应用通过全球 CDN 边缘节点分发，自动选择最近的节点，确保全球用户快速访问。

**Serverless Functions** — 提供 Serverless 函数能力，无需管理服务器即可运行后端 API 逻辑，支持 Node.js、Go、Python 等运行时。

**框架一级支持** — 对 Next.js、Nuxt、SvelteKit、Astro 等主流前端框架提供优化支持，零配置部署。

## 适用场景

**个人项目部署** — 将个人博客、作品集或 Side Project 部署到 Vercel，免费额度足够使用，自动 HTTPS 域名。

**前端团队协作** — 团队使用 Vercel 的 Preview 部署功能，每次 Pull Request 自动生成预览链接，方便代码审查和测试。

**全栈应用开发** — 利用 Serverless Functions 在 Vercel 上运行后端 API，实现前后端一体化的全栈应用部署。

## 使用技巧

**连接 GitHub 仓库** — 将代码仓库连接到 Vercel，后续每次推送代码都会自动部署，无需手动操作。

## 优缺点简评

**优势：** 部署体验极佳、全球CDN快速、Serverless函数、框架支持好、免费额度充足。

**不足：** 免费版有限制（带宽、构建时间）、锁定Vercel生态、国内访问速度一般、Serverless有冷启动。
