## 工具简介

Listary 是一款 Windows 全局文件搜索和启动增强工具，以极快的搜索速度和便捷的操作方式著称。双击 Ctrl 键即可在任何界面呼出搜索框，输入关键词即可秒搜文件、文件夹和应用程序。支持快捷命令（如"google 关键词"直接搜索）、自定义关键词和丰富的文件操作（复制、移动、删除等）。Listary 与 Windows 资源管理器深度集成，在任何文件对话框中都可以使用搜索功能，大幅提升 Windows 文件管理效率。

## 核心功能

**全局快捷搜索** — 双击 Ctrl 键即可在任何界面呼出搜索框，输入关键词秒搜文件、文件夹和应用程序，无需打开特定窗口。

**文件对话框集成** — 深度集成 Windows 文件打开/保存对话框，在"打开文件"和"保存文件"窗口中也可以使用 Listary 搜索。

**快捷命令** — 支持自定义快捷命令（如输入"google 关键词"直接在浏览器中搜索，"ip"查看本机 IP 地址），提升操作效率。

**文件操作** — 搜索到文件后可以直接进行复制、移动、删除、重命名等操作，还可以快速打开文件所在目录。

## 适用场景

**快速启动应用** — 双击 Ctrl 输入应用名称首字母即可快速启动应用程序，比在开始菜单中翻找快得多。

**文件对话框快速定位** — 在"打开文件"对话框中使用 Listary 搜索快速定位目标文件夹，省去逐级点击的麻烦。

**文件管理效率提升** — 配合快捷命令和文件操作功能，将常用的文件管理操作简化为几个关键词输入。

## 使用技巧

**配置常用快捷命令** — 在设置中添加常用的快捷命令（如打开常用文件夹、搜索网站等），日常使用效率翻倍。

## 优缺点简评

**优势：** 搜索速度极快、全局快捷键便捷、文件对话框集成、快捷命令实用、操作效率高。

**不足：** 免费版功能有限（如缺少网络搜索）、高级功能需付费、仅支持Windows、学习成本略高。
