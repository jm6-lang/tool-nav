## 工具简介

Netlify 是一款领先的现代 Web 项目托管和部署平台，专注于静态网站和 Jamstack 架构应用。它提供从 Git 仓库到全球 CDN 的自动化部署流水线，支持 Next.js、Gatsby、Hugo、Astro 等主流前端框架。Netlify 还提供 Serverless Functions、表单处理、身份认证、分割测试等后端即服务功能，让前端开发者无需管理服务器即可构建全栈应用。

## 核心功能

**Git 自动化部署** — 关联 GitHub、GitLab 或 Bitbucket 仓库后，每次代码推送自动触发构建和部署，支持预览部署和分支部署。

**全球 CDN 与边缘网络** — 网站内容自动分发到全球边缘节点，配合智能路由和自动 HTTPS，确保全球用户快速访问。

**Serverless Functions** — 内置基于 AWS Lambda 的无服务器函数，支持 JavaScript/TypeScript 和 Go，可直接在项目中编写后端 API 逻辑。

**表单处理与身份认证** — 内置表单收集和存储功能，支持邮件通知和 Webhook；提供开箱即用的身份认证服务，支持 OAuth 和自定义认证。

## 适用场景

**静态网站与博客部署** — 个人博客、文档站点和营销页面等静态网站，通过 Netlify 实现一键部署和全球加速，免费套餐即可满足需求。

**Jamstack 全栈应用** — 使用 Next.js、Nuxt.js 等框架构建的全栈应用，结合 Netlify Functions 实现后端逻辑，无需单独管理服务器。

## 使用技巧

**利用分支预览部署** — 每个 Pull Request 自动生成预览 URL，团队成员可在预览环境中查看和测试变更，合并后再部署到生产环境。

**配置重定向和头信息** — 通过 netlify.toml 配置文件统一管理页面重定向规则和自定义 HTTP 头，实现 SEO 优化和安全策略配置。

## 优缺点简评

**优势：** 部署流程极简、免费套餐功能丰富、全球 CDN 性能优秀、内置后端服务完善、开发者体验出色。

**不足：** 构建时间有限制（免费版 300 分钟/月）、Serverless Functions 有执行时间限制、自定义服务器配置灵活性较低。
