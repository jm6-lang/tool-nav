## 工具简介

SpaceSniffer 是一款轻量级的磁盘空间可视化分析工具，采用创新的 Treemap（树状图）可视化技术，将磁盘上的文件和文件夹以大小成比例的方块形式展示。用户可以一眼看出哪些目录和文件占用了最多的空间，支持动态扫描动画、文件过滤和交互式浏览。SpaceSniffer 为绿色便携版，无需安装即可使用。

## 核心功能

**可视化磁盘分析** — 以动态方块图展示磁盘空间使用情况，方块大小与文件/文件夹大小成正比，不同颜色区分文件类型，一目了然。

**交互式浏览** — 支持双击方块深入查看子目录内容，右键可直接打开文件或所在文件夹，悬停显示文件详细信息和路径。

**文件过滤与搜索** — 内置强大的过滤功能，可按文件类型、大小、日期等条件过滤显示，支持正则表达式搜索特定文件。

**多视图与对比** — 支持对同一磁盘进行多次扫描并对比结果，快速发现空间变化，也可同时打开多个磁盘分区进行对比分析。

## 适用场景

**磁盘空间清理** — 当磁盘空间不足时，使用 SpaceSniffer 快速定位占用大量空间的文件和目录，有针对性地清理释放空间。

**查找大文件** — 在海量文件中快速找出体积最大的文件，如旧的视频文件、大型日志或缓存目录，便于整理和归档。

## 使用技巧

**使用过滤功能精准定位** — 输入 `*.log` 可只显示日志文件，输入 `>1GB` 可只显示大于 1GB 的文件，快速锁定清理目标。

**以管理员身份运行** — 扫描 C 盘时建议以管理员身份运行，否则部分系统保护的目录可能无法正确读取大小。

## 优缺点简评

**优势：** 可视化效果直观、绿色便携无需安装、扫描速度快、交互操作便捷、完全免费。

**不足：** 界面风格较为简朴、不支持文件直接删除操作（需手动定位后删除）、长时间未更新。
