## 工具简介

Mozilla Firefox 是由 Mozilla 基金会维护的开源网页浏览器，采用自研的 Gecko 渲染引擎。Firefox 以用户隐私保护为核心设计理念，内置增强型跟踪保护（Enhanced Tracking Protection）和容器标签页（Container Tabs）等隐私功能。高度可定制的界面和丰富的扩展生态满足不同用户的需求。作为独立于 Chromium 的浏览器，Firefox 在推动 Web 标准多元化和用户选择权方面发挥着重要作用。

## 核心功能

**增强型跟踪保护** — 默认开启跟踪器拦截，自动阻止社交媒体跟踪器、跨站跟踪 Cookie 和加密货币挖矿脚本，保护用户隐私。

**容器标签页** — 支持将不同网站的 Cookie 和数据隔离到不同容器中（如工作、个人、购物），防止跨站追踪和身份关联。

**高度可定制** — 界面布局、工具栏按钮、主题外观均可自定义，about:config 页面提供数百项高级设置供高级用户调整。

**扩展生态** — 拥有丰富的 Firefox Add-ons 扩展生态，虽然数量不及 Chrome Web Store，但质量普遍较高且隐私审查严格。

## 适用场景

**隐私保护浏览** — 注重隐私的用户使用 Firefox 的增强型跟踪保护和容器标签页，有效防止在线追踪和隐私泄露。

**Web 开发调试** — Firefox 的开发者工具在 CSS 调试和性能分析方面有独到之处，部分功能（如 CSS Grid 调试器）优于 Chrome DevTools。

**多账号管理** — 利用容器标签页同时登录同一网站的不同账号（如多个社交媒体账号），互不干扰。

## 使用技巧

**配置容器标签页** — 安装 Multi-Account Containers 扩展，为工作、个人、购物等场景创建独立的浏览容器。

## 优缺点简评

**优势：** 隐私保护出色、开源透明、高度可定制、容器标签页实用、独立于Chromium生态。

**不足：** 性能不及Chrome、部分网站兼容性问题、扩展数量较少、移动端体验一般。
