## 工具简介

Aria2 是一款轻量级、多协议、多源的命令行下载工具，支持 HTTP/HTTPS、FTP、BitTorrent 和磁力链接等主流下载协议。Aria2 以其极低的资源占用和出色的下载性能著称，特别适合在服务器、NAS、路由器等资源受限的环境中使用。虽然 Aria2 本身是命令行工具，但社区开发了多种图形前端（如 AriaNg、Aria2 WebUI），让普通用户也能方便地使用。Aria2 完全开源免费，是技术用户和服务器管理员的下载利器。

## 核心功能

**多协议支持** — 同时支持 HTTP/HTTPS、FTP、BitTorrent、Metalink 等多种下载协议，一个工具覆盖几乎所有下载需求。
**多线程与多源下载** — 支持多线程分块下载加速，BT 下载支持多 Tracker 和 DHT/PEX，最大化利用带宽资源。
**RPC 远程控制** — 提供 JSON-RPC 和 XML-RPC 接口，支持通过 Web 界面或脚本远程控制下载任务，方便集成到自动化工作流中。

## 适用场景

**服务器与 NAS 下载** — 在 Linux 服务器或 NAS 设备上部署 Aria2，配合 Web 前端实现远程下载管理，是 PT 玩家和 NAS 用户的标配方案。
**BT/磁力下载** — 作为轻量级 BT 客户端使用，资源占用远低于 qBittorrent 等图形化客户端，适合后台长期挂机下载。

## 使用技巧

**配合 AriaNg 使用** — AriaNg 是最流行的 Aria2 Web 前端，部署简单，界面友好，通过浏览器即可远程管理所有下载任务。

## 优缺点简评

**优势：** 完全开源免费、资源占用极低、多协议支持、RPC 远程控制、社区生态丰富。

**不足：** 无图形界面（需配合前端使用）、配置需要命令行操作、新手上手门槛较高、不支持浏览器视频抓取。
