## 工具简介

Everything 是 Windows 平台最快的文件搜索工具，由 Voidtools 开发。Everything 利用 NTFS 文件系统的 USN Journal（更新序列号日志）建立文件名索引，搜索时直接查询索引而非遍历磁盘，因此可以在毫秒级内从数百万个文件中定位目标文件。支持正则表达式、文件类型过滤、大小和日期筛选、HTTP/FTP 服务器分享等高级功能，是 Windows 用户提升文件查找效率的必备工具。

## 核心功能

**即时搜索** — 基于 NTFS 索引实现毫秒级搜索，输入关键词瞬间显示所有匹配结果，百万级文件也能秒级定位。

**正则表达式** — 支持正则表达式搜索，可以进行复杂的模式匹配和条件搜索，满足高级用户的精确搜索需求。

**高级过滤** — 支持按文件类型、大小、日期、路径等条件过滤，支持布尔运算符（AND/OR/NOT）组合多个条件。

**HTTP/FTP 服务器** — 内置 HTTP 和 FTP 服务器功能，可以通过浏览器远程搜索和下载文件，方便局域网文件共享。

## 适用场景

**快速定位文件** — 忘记文件放在哪里时，用 Everything 搜索文件名或关键词，瞬间找到，比 Windows 自带搜索快千百倍。

**程序员代码搜索** — 搜索特定扩展名的文件（如 *.py、*.js），快速定位项目中的代码文件。

**局域网文件共享** — 开启 HTTP 服务器功能，局域网内其他设备通过浏览器即可搜索和下载文件。

## 使用技巧

**使用快捷键呼出** — 设置全局快捷键（如 Ctrl+Alt+F）随时呼出 Everything 搜索框，比打开开始菜单搜索更高效。

## 优缺点简评

**优势：** 搜索速度极快、功能强大、支持正则表达式、HTTP服务器实用、完全免费。

**不足：** 仅支持NTFS格式、只索引文件名不索引文件内容、界面较简陋、不支持非NTFS分区。
