## 工具简介

CPU-Z是一款经典的硬件信息检测工具，界面简洁直观，可以详细显示CPU、内存、主板和显卡的规格参数。工具提供实时频率、电压、时序等详细信息，是硬件玩家检测配置、验证产品真伪和进行跑分对比的必备工具。绿色免安装，单文件即可运行。

## 核心功能

**CPU详细信息** — 显示处理器名称、核心数、频率、缓存等全部规格。

**内存信息查看** — 检测内存类型、容量、频率、时序等参数。

**主板信息** — 显示主板型号、芯片组、BIOS版本等主板信息。

**显卡信息** — 提供显卡核心、显存容量等GPU详细信息。

## 适用场景

**配置信息检测** — 购买电脑后使用CPU-Z验证配置是否与宣传一致。

**硬件性能评估** — 查看硬件参数，为升级或对比提供数据参考。

**游戏需求匹配** — 查看配置后判断电脑能否运行特定游戏。

## 使用技巧

查看实时频率 — 切换到 Bench 标签可以看到处理器在负载下的真实频率。

保存硬件报告 — 可以截图或使用软件自带的报告功能保存检测结果。

**优势：** 完全免费绿色免安装、界面简洁、信息全面、硬件玩家必备。
